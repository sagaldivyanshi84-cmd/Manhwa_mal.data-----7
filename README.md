## Manhwa SQL Project ---7

## About
This project is based on a Manhwa dataset and uses SQL to analyze different aspects of popular manhwa series such as genres, ratings, authors, release years, readers, popularity, and status.  The project helps practice real-world SQL concepts including filtering, aggregate functions, CET and data analysis.

## Objectives
- Analyze popular manhwa titles
- Find highest-rated and most-read series
- Explore genre trends
- Practice SQL queries on entertainment data
- Improve database management and analytical skills
